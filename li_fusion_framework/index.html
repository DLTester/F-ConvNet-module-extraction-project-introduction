Li-Fusion Framework 模块文档¶

本模块定义了 EPNet 风格的 LiDAR-Image 融合框架 (Li-Fusion Framework)，
整合了几何对齐与特征融合两个阶段，实现点云与图像的端到端特征融合。

模块概览¶

模块路径：li_fusion_framework

核心功能包括： - 点云与图像特征提取（Backbone 模块）； - 点云到图像的几何投影； - 特征网格采样； - 特征融合（加权 + 门控 + 拼接）。

class LIFusionCore(nn.Module):
    """
    Li-Fusion 核心融合模块：
    对齐点云和图像通道后执行加权融合。
    """

    def __init__(self, in_p: int, in_i: int, fuse_d: int):
        """
        Args:
            in_p (int): 点云特征通道数。
            in_i (int): 图像特征通道数。
            fuse_d (int): 融合后目标通道数。
        """
        super().__init__()
        self.fc_p = FCAlignP(in_p, fuse_d, use_bn=True, act=True)      # 点云通道对齐层
        self.fc_i = FCAlignI(in_i, fuse_d, use_bn=True, act=True)      # 图像通道对齐层
        self.fc_w = FeatureWeightFC(in_dim=fuse_d, out_dim=1, use_bn=False)  # 权重生成层

    def forward(self, Fp: torch.Tensor, Fi_point: torch.Tensor) -> Tuple[torch.Tensor, Dict[str, torch.Tensor]]:
        """
        执行跨模态融合。

        Args:
            Fp (torch.Tensor): 点云特征，形状 (B, Dp, N)。
            Fi_point (torch.Tensor): 图像采样特征，形状 (B, Ci, N)。

        Returns:
            Tuple[torch.Tensor, Dict[str, torch.Tensor]]:
                - F_LI: 融合特征 (B, 2*fuse_d, N)
                - info: 信息字典（含中间结果与权重），键包括：
                    {"w": (B,1,N), "Fp_": (B,fuse_d,N), "Fi_": (B,fuse_d,N),
                     "H": (B,fuse_d,N), "Fi_gated": (B,fuse_d,N)}
        """
        Fp_ = self.fc_p(Fp)  # 点云特征通道对齐
        Fi_ = self.fc_i(Fi_point)  # 图像特征通道对齐
        H = torch.ops.li_fusion.li_fusion_add(Fp_, Fi_, apply_tanh=True)  # 融合特征相加 + 激活
        w = self.fc_w(H)  # 生成融合权重
        Fi_gated = torch.ops.li_fusion.li_fusion_muti(Fi_, w, sigmoid_y=False)  # 权重门控图像特征
        F_LI = torch.ops.li_fusion.li_fusion_concat([Fp_, Fi_gated], dim=1)  # 拼接融合输出

        info: Dict[str, torch.Tensor] = {
            "w": w, "Fp_": Fp_, "Fi_": Fi_, "H": H, "Fi_gated": Fi_gated
        }
        return F_LI, info

class LIFusionPipeline(nn.Module):
    """
    Li-Fusion 融合流程管线 / Li-Fusion pipeline.

    负责：几何映射 → 图像采样到点 → Backbone 提特征 → 核心融合。
    注意：本版本在 __init__ 中**直接实例化 fusion_core**，不再懒构造。
    """

    def __init__(self,
                 lidar_backbone: LiDARBackbone,
                 image_backbone: ImageBackbone,
                 in_p: int,
                 in_i: int,
                 fuse_d: int) -> None:
        """
        Args:
            lidar_backbone (LiDARBackbone): 点云特征提取模块.
            image_backbone (ImageBackbone): 图像特征提取模块.
            in_p (int): 点云特征通道数 Dp（传给融合核心）.
            in_i (int): 图像特征通道数 Ci（传给融合核心）.
            fuse_d (int): 融合/对齐维度.
        """
        super().__init__()
        self.lidar_backbone = lidar_backbone                 # 保存骨干
        self.image_backbone = image_backbone                 # 保存骨干
        self.fuse_d = fuse_d                                 # 融合维度
        self.fusion_core = LIFusionCore(in_p=in_p,           # 直接实例化
                                        in_i=in_i,
                                        fuse_d=fuse_d)

    def forward(self,
                pointcloud: torch.Tensor,
                image: torch.Tensor,
                R: torch.Tensor,
                t: torch.Tensor,
                K: torch.Tensor) -> Dict[str, torch.Tensor]:
        """
        执行完整 Li-Fusion 前向流程 / Full forward.

        Args:
            pointcloud (torch.Tensor): 点云 (B, N, C>=3).
            image (torch.Tensor): 图像 (B, 3, H, W).
            R (torch.Tensor): 相机旋转矩阵.
            t (torch.Tensor): 相机平移向量.
            K (torch.Tensor): 相机内参矩阵.

        Returns:
            Tuple[torch.Tensor, Dict[str, torch.Tensor]]:
                - F_LI: 融合特征 (B, 2*fuse_d, N)
                - info: 信息字典，至少包含 {"mask": (B,N), "w": (B,1,N)}，
                        并合并融合核心的中间结果（"Fp_","Fi_","H","Fi_gated"）
        """
        B, _, H, W = image.shape  # 尺度
        xyz = torch.ops.li_fusion.lidar_point_extractor(pointcloud)
        uv, depth, _M = torch.ops.li_fusion.lidar_to_image_projector(xyz, R, t, K)  # 投影到像素平面
        _xy_norm, grid, mask = torch.ops.li_fusion.grid_generator(W, H, uv, depth)  # 生成采样网格与掩码

        fmap = self.image_backbone(image)  # (B, Ci, H, W)
        Fi_point = torch.ops.li_fusion.image_sampler(fmap, grid)  # (B, Ci, N)：按点采样
        Fp = self.lidar_backbone(pointcloud)  # (B, Dp, N)：点云特征

        F_LI, core_info = self.fusion_core(Fp, Fi_point)  # 融合核心：主特征+字典
        info: Dict[str, torch.Tensor] = {"mask": mask, **core_info}  # 合并掩码
        return F_LI, info
