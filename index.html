EPNet Li-Fusion 模块文档¶

本项目基于 EPNet（Enhancing Point Cloud and Image Fusion for 3D Object Detection）
独立实现 Li-Fusion 跨模态融合机制，用于点云与图像的联合特征提取。

模块组成¶

  • 几何阶段（Geometry Stage）
    实现点云投影、网格生成、图像采样等几何对应操作。

  • 融合阶段（Fusion Stage）
    包含 Li-Fusion 融合算子（Add / Multiply / Concat）与特征加权模块。

  • ⚙框架核心（LIFusionPipeline）
    将点云与图像特征通过统一接口进行前向融合与可视化输出。
