Geometry Stage 模块¶

模块简介¶

Geometry Stage（几何阶段）是 Li-Fusion 框架的第一阶段，负责将 LiDAR 点云与图像在几何空间上对齐。
该阶段由一系列自定义算子（Custom Ops）组成，完成从三维点到二维像素的映射，为后续融合阶段提供特征级对齐的数据基础。

模块结构¶

算子名称 功能说明
lidar_point_extractor 从点云中提取 (x, y, z) 三维坐标。
lidar_to_image_projector 利用相机外参 (R, t) 与内参 (K) 将 LiDAR 点投影到图像平面，生成像素坐标 (u, v) 与深度 depth。
grid_generator 将像素坐标 (u, v) 转换为 grid_sample 可用的归一化采样网格 grid，并生成有效点掩码 mask。
image_sampler 根据采样网格 grid，在图像特征图上执行双线性插值采样，提取逐点图像特征，实现点云-图像对齐。

这些算子共同完成以下数据流：

LiDAR点云 → (提取坐标) → lidar_point_extractor → (投影至图像平面) → lidar_to_image_projector → (生成归一化网格) → grid_generator → (采样图像特征) → image_sampler

输出结果包括： - 对齐后的像素坐标 uv - 深度图 depth - 有效掩码 mask - 对应的逐点图像特征 Fi_point

1. lidar_point_extractor¶

提取点云坐标 (x, y, z)，为后续投影提供输入。

lidar_point_extractor(pointcloud) ¶

Li-Fusion 模块中的点云坐标提取算子。

从原始点云数据中提取三维坐标 (x, y, z)，用于后续几何投影与特征对齐。 输入的点云可能包含额外信息（如强度、反射率等），此函数仅保留前三个坐标通道。

    原始点云输入，形状 (B, N, C>=3)。 - B: 批大小 (batch size) - N: 点数 (number of points) - C: 每个点的通道数，至少包含 (x, y, z)
Returns:
  • Tensor –

    torch.Tensor: 点云坐标张量 xyz，形状 (B, N, 3)。 - 第 0~2 通道分别对应 (x, y, z)

@torch.library.custom_op("li_fusion::lidar_point_extractor", mutates_args=[])
def lidar_point_extractor(pointcloud: torch.Tensor) -> torch.Tensor:
    """
    Li-Fusion 模块中的点云坐标提取算子。

    从原始点云数据中提取三维坐标 (x, y, z)，用于后续几何投影与特征对齐。
    输入的点云可能包含额外信息（如强度、反射率等），此函数仅保留前三个坐标通道。

    Args:
        pointcloud (torch.Tensor): 原始点云输入，形状 (B, N, C>=3)。
            - B: 批大小 (batch size)
            - N: 点数 (number of points)
            - C: 每个点的通道数，至少包含 (x, y, z)

    Returns:
        torch.Tensor: 点云坐标张量 xyz，形状 (B, N, 3)。
            - 第 0~2 通道分别对应 (x, y, z)
    """
    xyz = pointcloud[..., 0:3].contiguous().clone()  # 提取 (x, y, z)，保持内存连续
    return xyz

2. lidar_to_image_projector¶

根据相机参数 (R, t, K) 计算点云在图像平面的投影结果。

lidar_to_image_projector(lidar_points, R, t, K) ¶

Li-Fusion 模块中的点云到图像坐标投影算子。

将三维点云坐标 (x, y, z) 投影到相机平面，得到像素坐标 (u, v) 与深度值 depth。 该算子执行矩阵运算 M = K [R|t]，完成点云从 LiDAR 坐标系到图像坐标系的转换。

  • R (Tensor) –

    相机外参旋转矩阵，形状 (B, 3, 3)

  • t (Tensor) –

    相机外参平移向量，形状 (B, 3, 1)

  • K (Tensor) –

    相机内参矩阵，形状 (B, 3, 3)
Returns:
  • List[Tensor] –

    List[torch.Tensor]: - uv: 投影到图像平面的像素坐标 (B, N, 2) - depth: 对应深度值 (B, N, 1) - M: 相机投影矩阵 K [R|t] (B, 3, 4)

@torch.library.custom_op("li_fusion::lidar_to_image_projector", mutates_args=[])
def lidar_to_image_projector(
    lidar_points: torch.Tensor,  # (B, N, 3)
    R: torch.Tensor,             # (B, 3, 3)
    t: torch.Tensor,             # (B, 3, 1)
    K: torch.Tensor              # (B, 3, 3)
) -> List[torch.Tensor]:
    """
    Li-Fusion 模块中的点云到图像坐标投影算子。

    将三维点云坐标 (x, y, z) 投影到相机平面，得到像素坐标 (u, v) 与深度值 depth。
    该算子执行矩阵运算 M = K [R|t]，完成点云从 LiDAR 坐标系到图像坐标系的转换。

    Args:
        lidar_points (torch.Tensor): 点云坐标，形状 (B, N, 3)
            - B: 批大小
            - N: 点数
            - 每个点包含 (x, y, z)
        R (torch.Tensor): 相机外参旋转矩阵，形状 (B, 3, 3)
        t (torch.Tensor): 相机外参平移向量，形状 (B, 3, 1)
        K (torch.Tensor): 相机内参矩阵，形状 (B, 3, 3)

    Returns:
        List[torch.Tensor]:
            - **uv**: 投影到图像平面的像素坐标 (B, N, 2)
            - **depth**: 对应深度值 (B, N, 1)
            - **M**: 相机投影矩阵 K [R|t] (B, 3, 4)
    """
    B, N, _ = lidar_points.shape
    device = lidar_points.device

    # Step 1: 齐次坐标
    ones = torch.ones((B, N, 1), device=device)
    lidar_homo = torch.cat([lidar_points, ones], dim=-1)  # (B, N, 4)

    # Step 2: 投影矩阵 M = K [R|t]
    Rt = torch.cat([R, t], dim=-1)                       # (B, 3, 4)
    M = torch.bmm(K, Rt)                                 # (B, 3, 4)

    # Step 3: 投影到相机平面
    uv_depth = torch.bmm(lidar_homo, M.transpose(1, 2))  # (B, N, 3)

    # Step 4: 提取 (u, v) 与 depth
    depth = uv_depth[..., 2:3]                           # (B, N, 1)
    u = uv_depth[..., 0] / (depth[..., 0] + 1e-6)        # (B, N)
    v = uv_depth[..., 1] / (depth[..., 0] + 1e-6)
    uv = torch.stack([u, v], dim=-1)                     # (B, N, 2)

    return [uv.clone(), depth.clone(), M.clone()]

3. grid_generator¶

生成归一化采样网格 grid:(B,1,N,2) 和有效掩码 mask:(B,N)。

grid_generator(img_w, img_h, uv, depth) ¶

Li-Fusion 模块中的采样网格生成算子。

根据点云投影后的像素坐标 (u, v) 与深度值，生成归一化采样网格、归一化坐标与有效掩码。 该算子用于从图像特征图中按点云投影位置采样特征，是 Li-Fusion 模块中几何对齐的关键步骤。

@torch.library.custom_op("li_fusion::grid_generator", mutates_args=[])
def grid_generator(img_w: int, img_h: int, uv: torch.Tensor, depth: torch.Tensor) -> List[torch.Tensor]:
    """
    Li-Fusion 模块中的采样网格生成算子。

    根据点云投影后的像素坐标 (u, v) 与深度值，生成归一化采样网格、归一化坐标与有效掩码。
    该算子用于从图像特征图中按点云投影位置采样特征，是 Li-Fusion 模块中几何对齐的关键步骤。

    Args:
        img_w (int): 图像宽度 W。
        img_h (int): 图像高度 H。
        uv (torch.Tensor): 点云投影到图像平面的像素坐标，形状为 (B, N, 2)。
            - B: 批大小
            - N: 点数量
            - 每个点包含像素坐标 (u, v)
        depth (torch.Tensor): 投影点对应的深度值，形状为 (B, N, 1)。

    Returns:
        List[torch.Tensor]: 列表 [xy_norm, grid, mask]。
            - **xy_norm** (`torch.Tensor`): 归一化像素坐标 (-1~1)，形状 (B, N, 2)。
            - **grid** (`torch.Tensor`): 用于 `torch.nn.functional.grid_sample` 的采样网格，形状 (B, 1, N, 2)。
            - **mask** (`torch.Tensor`): 有效点掩码，True 表示点在图像范围内，形状 (B, N)。
    """
    W, H = img_w, img_h

    # 避免直接对 uv 的视图进行后续计算，先拷贝一份（可选但更稳）
    u = uv[..., 0].contiguous()         # 或 uv[..., 0].clone()
    v = uv[..., 1].contiguous()

    valid_depth = (depth[..., 0] > 0)
    valid_u = (u >= 0) & (u <= (W - 1))
    valid_v = (v >= 0) & (v <= (H - 1))
    mask = (valid_depth & valid_u & valid_v)

    u_norm = (u / (W - 1.0)) * 2.0 - 1.0
    v_norm = (v / (H - 1.0)) * 2.0 - 1.0
    xy_norm = torch.stack([u_norm, v_norm], dim=-1).contiguous()
    grid = xy_norm.unsqueeze(1).contiguous()

    # 关键：返回前做 clone，确保与输入不共享存储
    return [xy_norm.clone(), grid.clone(), mask.clone()]

4. image_sampler¶

利用 grid_sample 在图像特征图上采样每个点的图像特征，实现几何级融合。

@torch.library.custom_op("li_fusion::image_sampler", mutates_args=[])
def image_sampler(feature_map: torch.Tensor, grid: torch.Tensor) -> torch.Tensor:
    """
    Li-Fusion 模块中的图像特征采样算子。

    基于 `grid_sample` 操作，从图像特征图中按指定采样网格提取逐点特征。
    通常与 `grid_generator` 配合使用，实现点云到图像特征的对齐。

    Args:
        feature_map (torch.Tensor): 图像特征图，形状 (B, C, H, W)。
        grid (torch.Tensor): 采样网格，形状 (B, 1, N, 2)，来自 `grid_generator`。

    Returns:
        torch.Tensor: 采样到的逐点图像特征，形状 (B, C, N)。
    """
    sampled = F.grid_sample(feature_map, grid, align_corners=True)  # (B, C, 1, N)
    sampled = sampled.squeeze(2).contiguous()
    return sampled.clone()
