Fusion Stage 模块¶

模块说明¶

融合阶段（Fusion Stage）负责点云与图像特征的跨模态融合。 其主要功能包括特征对齐、特征加权与融合操作。

本阶段包含以下子模块：

  • li_fusion_add — 特征相加并可选 tanh 激活；
  • li_fusion_muti — 特征门控/加权乘法；
  • li_fusion_concat — 通道拼接融合；
  • feature_weight_fc — 通道注意力权重计算；
  • align_fusion_stage — 点云与图像对齐模块：
  • fc_align_i — 图像通道对齐；
  • fc_align_p — 点云通道对齐。

基础融合算子¶

li_fusion_add(Fp, Fi, apply_tanh=True) ¶

Li-Fusion 模块中的跨模态特征融合算子（加法 + 可选 Tanh 激活）。

对点云特征 Fp 与图像特征 Fi 进行逐元素相加融合，可选择是否使用 Tanh 激活进行非线性约束。 该算子用于 Li-Fusion 核心模块中的主融合步骤。

  • Fi (Tensor) –

    图像特征张量，形状 (B, D, N)。 - 与 Fp 具有相同维度

  • apply_tanh (bool, default: True ) –

    是否应用 tanh 激活，默认 True。
Returns:
  • Tensor –

    torch.Tensor: 融合输出特征张量，形状 (B, D, N)。

@torch.library.custom_op("li_fusion::li_fusion_add", mutates_args=[])
def li_fusion_add(Fp: torch.Tensor, Fi: torch.Tensor, apply_tanh: bool = True) -> torch.Tensor:
    """
    Li-Fusion 模块中的跨模态特征融合算子（加法 + 可选 Tanh 激活）。

    对点云特征 Fp 与图像特征 Fi 进行逐元素相加融合，可选择是否使用 Tanh 激活进行非线性约束。
    该算子用于 Li-Fusion 核心模块中的主融合步骤。

    Args:
        Fp (torch.Tensor): 点云特征张量，形状 (B, D, N)。
            - B: 批大小
            - D: 通道数
            - N: 点数量
        Fi (torch.Tensor): 图像特征张量，形状 (B, D, N)。
            - 与 Fp 具有相同维度
        apply_tanh (bool): 是否应用 tanh 激活，默认 True。

    Returns:
        torch.Tensor: 融合输出特征张量，形状 (B, D, N)。
    """
    out = Fp + Fi
    if apply_tanh:
        out = torch.tanh(out)
    return out.clone()  # 避免 alias 报错

li_fusion_muti(x, y, sigmoid_y=False) ¶

Li-Fusion 模块中的特征门控乘法算子（Multiply-Gate Operator）。

实现跨模态融合中的逐点加权操作：对输入特征 x 与权重特征 y 逐元素相乘。 当启用 sigmoid_y 时，会对 y 进行 Sigmoid 压缩到 [0, 1] 区间，用于门控控制。

Parameters:
  • x (Tensor) –

    主输入特征张量，形状 (B, D, N)。 - B: 批大小 - D: 通道数 - N: 点数量

  • y (Tensor) –

    权重特征张量，形状 (B, D, N) 或 (B, 1, N)。 - 当 y 为单通道时，会广播匹配 x 的形状

  • sigmoid_y (bool, default: False ) –

    是否对 y 施加 Sigmoid 激活（默认 False）。
Returns:
  • Tensor –

    torch.Tensor: 加权融合结果张量，形状与 x 相同 (B, D, N)。

def li_fusion_muti(x: torch.Tensor, y: torch.Tensor, sigmoid_y: bool = False) -> torch.Tensor:
    """
    Li-Fusion 模块中的特征门控乘法算子（Multiply-Gate Operator）。

    实现跨模态融合中的逐点加权操作：对输入特征 x 与权重特征 y 逐元素相乘。
    当启用 `sigmoid_y` 时，会对 y 进行 Sigmoid 压缩到 [0, 1] 区间，用于门控控制。

    Args:
        x (torch.Tensor): 主输入特征张量，形状 (B, D, N)。
            - B: 批大小
            - D: 通道数
            - N: 点数量
        y (torch.Tensor): 权重特征张量，形状 (B, D, N) 或 (B, 1, N)。
            - 当 y 为单通道时，会广播匹配 x 的形状
        sigmoid_y (bool): 是否对 y 施加 Sigmoid 激活（默认 False）。

    Returns:
        torch.Tensor: 加权融合结果张量，形状与 x 相同 (B, D, N)。
    """
    if sigmoid_y:
        y = torch.sigmoid(y)
    out = x * y
    return out.clone()

li_fusion_concat(tensors, dim=1) ¶

Li-Fusion 模块中的特征拼接算子（Concat Operator）。

class FeatureWeightFC(nn.Module):
    """
    Li-Fusion 模块中的特征权重生成层（Feature Weight Fully Connected Layer）。

    根据融合特征 H 计算通道权重，用于控制图像或点云特征的融合强度。
    通常在 Li-Fusion 核心模块中接收融合特征作为输入，输出权重向量 (B, 1, N)。
    """

    def __init__(self, in_dim: int, out_dim: int = 1, use_bn: bool = False):
        """
        Args:
            in_dim (int): 输入特征通道数（融合特征维度）。
            out_dim (int): 输出通道数（默认 1，对应单通道权重）。
            use_bn (bool): 是否使用 BatchNorm1d。
        """
        super().__init__()
        layers = [nn.Conv1d(in_dim, out_dim, 1, bias=not use_bn)]
        if use_bn:
            layers.append(nn.BatchNorm1d(out_dim))
        self.fc = nn.Sequential(*layers)

    def forward(self, H: torch.Tensor) -> torch.Tensor:
        """
        计算特征权重。

        Args:
            H (torch.Tensor): 融合输入特征张量，形状 (B, D, N)。
                - B: 批大小
                - D: 通道数（in_dim）
                - N: 点数量

        Returns:
            torch.Tensor: 权重张量 w，形状 (B, 1, N)，
                值范围在 [0, 1]，用于加权融合。
        """
        w = self.fc(H)
        w = torch.sigmoid(w)
        return w

class FCAlignI(nn.Module):
    """
    图像特征通道对齐层（Image Feature Channel Alignment Layer）。

    用于将图像特征通道数调整至指定维度，以便与点云特征在融合阶段保持一致。
    采用 1×1 卷积实现通道映射，并可选 BatchNorm 与 ReLU 激活。
    """

    def __init__(self, in_i: int, out_d: int, use_bn: bool = True, act: bool = True):
        """
        Args:
            in_i (int): 输入图像特征通道数。
            out_d (int): 输出目标通道数（融合维度）。
            use_bn (bool): 是否启用 BatchNorm1d。
            act (bool): 是否启用 ReLU 激活。
        """
        super().__init__()
        layers = [nn.Conv1d(in_i, out_d, 1, bias=not use_bn)]
        if use_bn:
            layers.append(nn.BatchNorm1d(out_d))
        if act:
            layers.append(nn.ReLU(inplace=True))
        self.net = nn.Sequential(*layers)

    def forward(self, Fi: torch.Tensor) -> torch.Tensor:
        """
        执行图像特征通道映射。

        Args:
            Fi (torch.Tensor): 图像特征输入，形状 (B, Ci, N)。
                - B: 批大小
                - Ci: 输入通道数
                - N: 特征点数量（通常为投影点数）

        Returns:
            torch.Tensor: 通道对齐后的特征张量 (B, D, N)。
                - D: 输出通道数（out_d）
        """
        return self.net(Fi)
